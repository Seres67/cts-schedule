# cts-schedule

Prints next departures at selected CTS station.

## Prerequisites

- Go 1.23.2

## Usage

```bash
go build
./cts-times "Homme de Fer"
```

Returns:
```bash
21:56 Graffenstaden
21:57 Hoenheim Gare
21:58 Parc des Sports
22:01 Lingolsheim Tiergaertel
22:02 Comtes
22:02 Poteries
22:05 Gare Centrale
22:06 Kehl Rathaus
22:07 Neuhof R.Reuss
22:07 Place d'Islande
```
